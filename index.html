HTML ist eine textbasierte Programmiersprache mit der den Inhalt von Webseiten definieren kann.
CSS beschreibt, wie Elemente aussehen und kann ihre Eigenschaften bei bestimmten Events ändern (Subklassen, zB Maus über element, hover)
JavaScript macht Webseiten dynamisch, das heißt es wird zB bei einem Knopfdruck ein bestimmter Code ausgeführt.
