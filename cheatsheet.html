CSS Selectors

Selector Example Description
#id #firstname Selects the element with id="firstname"
.class .intro Selects all elements with class="intro".
element.class p.intro Selects only <p> elements with class="intro"
* * Selects all elements
Child Selector ul > li Selects all <li> elements that are direct children of <ul>.
Adjacent Sibling Selector h2 + p Selects the <p> element that directly follows an <h2>.
General Sibling Selector h2 ~ p Selects all <p> elements that are siblings of an <h2>.

URL



A URL's syntax typically looks like a URI (Uniform Resource Locator), which look like this:
[scheme]://[authority]/[path]?[query]#[fragment]

Querys usually look like this: &[query=value]&[second query=value]. The authority contains [userinfo]@[host]:[port]. The host is the domain, the userinfo may include content such as passwords and the port is a number.
